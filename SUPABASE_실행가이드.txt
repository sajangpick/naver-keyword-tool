====================================
🎯 Supabase SQL 실행 가이드
====================================

📝 실행 순서:
1. simple-analytics.sql
2. simple-error-logging.sql  
3. simple-monitoring.sql
4. supabase-rls-monitoring.sql ⭐ 중요!

====================================
실행 방법 (각 파일마다 반복)
====================================

1단계: Supabase 접속
   https://supabase.com → 로그인 → 프로젝트 선택

2단계: SQL Editor 열기
   왼쪽 메뉴에서 "SQL Editor" 클릭

3단계: SQL 파일 복사
   VS Code에서:
   - Ctrl + A (전체 선택)
   - Ctrl + C (복사)

4단계: Supabase에 붙여넣기
   - 입력창 클릭
   - Ctrl + V (붙여넣기)

5단계: 실행
   - "Run" 버튼 클릭 (또는 Ctrl + Enter)
   - "Success. No rows returned" 확인

6단계: 다음 파일로
   - 입력창 전체 지우기 (Ctrl + A → Delete)
   - 다음 SQL 파일로 이동하여 반복

====================================
✅ 성공 확인
====================================

모든 파일 실행 후:
→ Table Editor에서 테이블 확인
→ user_events, daily_stats 등 테이블 생김

====================================
❌ 에러 발생 시
====================================

"already exists" 에러:
   → 이미 있다는 뜻 (괜찮음, 다음 파일로)

"permission denied" 에러:
   → 프로젝트 소유자 계정인지 확인

기타 에러:
   → 에러 메시지 복사해서 AI에게 질문
