# 🚀 서버 시작 방법

## 방법 1: 배치 파일 사용 (가장 간단)

1. 프로젝트 폴더에서 `서버시작.bat` 파일을 **더블클릭**
2. 서버가 시작되면 브라우저에서 접속:
   ```
   http://localhost:3003/ChatGPT.html
   ```

## 방법 2: 터미널에서 직접 실행

### PowerShell 사용 시
```powershell
# 프로젝트 폴더로 이동
cd "C:\Users\오다쉐프\Desktop\사징픽_권순미 최종수정중"

# 서버 실행
node server.js
```

### 명령 프롬프트(cmd) 사용 시
```cmd
# 프로젝트 폴더로 이동
cd "C:\Users\오다쉐프\Desktop\사징픽_권순미 최종수정중"

# 서버 실행
node server.js
```

## 방법 3: VS Code 사용 시

1. VS Code에서 프로젝트 폴더 열기
2. 터미널 열기 (Ctrl + `)
3. 다음 명령어 실행:
   ```bash
   node server.js
   ```

## ✅ 서버가 실행되면

터미널에 다음과 같은 메시지가 보입니다:
```
🚀 통합 API 서버가 시작되었습니다!
🌐 서버 주소: http://0.0.0.0:3003
```

그런 다음 브라우저에서:
```
http://localhost:3003/ChatGPT.html
```
로 접속하세요.

## 🛑 서버 종료

터미널에서 `Ctrl + C`를 누르면 서버가 종료됩니다.

## ⚠️ 문제 해결

### "node를 찾을 수 없습니다" 오류
- Node.js가 설치되어 있지 않습니다
- [Node.js 다운로드](https://nodejs.org/) 후 설치하세요

### "포트가 이미 사용 중입니다" 오류
- 다른 프로그램이 3003 포트를 사용 중입니다
- 해당 프로그램을 종료하거나 `.env` 파일에 `PORT=3004` 추가

### "모듈을 찾을 수 없습니다" 오류
- 패키지가 설치되지 않았습니다
- 터미널에서 `pnpm install` 또는 `npm install` 실행
