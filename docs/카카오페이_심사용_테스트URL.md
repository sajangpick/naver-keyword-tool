> **작성일**: 2025년 11월 25일

---

## 📋 개요

사장픽 서비스는 카카오 로그인만 지원하는 구조로, 보안상의 이유로 로그인 없이는 결제 페이지에 접근할 수 없습니다.  
카카오페이 심사팀의 요청에 따라 **데모 모드**를 추가하여 로그인 없이도 결제 페이지를 확인할 수 있도록 했습니다.

---

## 🔗 테스트 URL

### 1. 플랜 선택 페이지 (메인)
```
https://sajangpick.co.kr/payment.html?demo=true
```

**설명**: 
- 모든 플랜(라이트, 스탠다드, 프로, 프리미엄)을 확인할 수 있습니다
- 플랜 카드를 클릭하면 해당 플랜의 결제 화면으로 이동합니다
- 로그인 없이 접근 가능합니다

## ✅ 데모 모드 특징

### 1. **로그인 불필요**
- 카카오 로그인 없이 바로 접근 가능
- 인증 절차 없이 결제 페이지 확인 가능

### 2. **데모 배너 표시**
- 페이지 상단에 주황색 배너가 표시됩니다
- "🔍 데모 모드: 카카오페이 심사용 테스트 페이지입니다" 메시지 표시

### 3. **결제 기능 비활성화**
- 실제 결제는 진행되지 않습니다
- 결제 버튼이 비활성화되어 있습니다
- 결제 방법 선택도 비활성화되어 있습니다

### 4. **실제 UI 확인 가능**
- 실제 결제 페이지와 동일한 디자인
- 플랜 정보, 가격, 토큰 정보 모두 표시
- 결제 방법 선택 UI 확인 가능 (카카오페이 포함)

## 📱 확인 가능한 기능

### ✅ 확인 가능한 항목
1. **플랜 선택 화면**
   - 4개 플랜 카드 (라이트, 스탠다드, 프로, 프리미엄)
   - 각 플랜의 가격 및 토큰 정보
   - 플랜 카드 클릭 시 결제 화면 이동

2. **결제 화면**
   - 선택한 플랜 정보 표시
   - 가격 및 토큰 정보
   - 결제 방법 선택 UI
     - 간편 결제: 토스페이, **카카오페이**, 네이버페이
     - 일반 결제: 신용/체크카드, 계좌이체, 휴대폰 결제
   - 카드 등록 기능 UI

3. **반응형 디자인**
   - 데스크톱, 태블릿, 모바일 모두 지원
   - 각 화면 크기에서 UI 확인 가능

### ❌ 작동하지 않는 항목 (데모 모드)
1. 실제 결제 진행 불가
2. 카드 등록 불가
3. 로그인 관련 기능 (데모 모드이므로 불필요)

## 🔒 보안 고려사항

1. **데모 모드 제한**
   - `?demo=true` 파라미터가 있을 때만 로그인 우회
   - 데모 모드에서는 실제 결제 API 호출 불가
   - 결제 버튼 및 결제 방법 선택 비활성화

2. **프로덕션 환경**
   - 일반 사용자는 `?demo=true` 없이 접근 시 로그인 필수
   - 데모 모드는 심사용으로만 사용 권장
