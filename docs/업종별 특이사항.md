## 🍽️ 음식점 - 우리 사이트가 직접 해주는 것(실행 패키지)

식당회원이 네이버 플레이스에 바로 적용할 수 있도록, 우리 사이트는 아래 산출물을 “복사/다운로드/바로가기”로 즉시 제공합니다. 복잡한 통합 없이도 실행만 하면 전환에 연결되도록 설계했습니다.

- 예약/대기 ON 지원
  - 예약·대기 안내문 자동 생성(피크타임/좌석수 반영) 3안
  - 직원 멘트 스크립트(계산대/대기 시 한 문장 안내)
  - POP·테이블텐트 PDF(예약/길찾기 QR 포함)
  - 네이버 관리자 바로가기 + 체크리스트(필수 항목 복사 버튼)

- 메뉴·가격 최신화(대표 10개) 보조
  - 메뉴 시트 템플릿(CSV/구글시트): 이름·가격·설명·해시태그 칼럼 자동 생성
  - 대표메뉴 우선순위 추천 + 설명문구/해시태그 자동 제안
  - 사진 일괄 리사이즈/압축(권장 비율·용량, 파일명 규칙 포함)

- 대표사진 5장(외관/내부/시그니처/단체좌석/주차) 촬영·정리 키트
  - 샷 리스트(모바일 체크리스트, 구도 예시/문구 샘플)
  - 자동 보정·크롭(권장 비율/썸네일 크롭)
  - 업로드 순서 가이드 + 사진별 설명문 복사

- 영수증리뷰 유도 캠페인
  - 리뷰 유도 멘트 5안(계산 멘트/영수증 하단 문구/문자 템플릿)
  - 리뷰 QR 포스터 PDF(플레이스/길찾기 QR)
  - 직원 공지 카드(도움 되는 리뷰 예시)
  - 캠페인 체크바(‘공지/포스터/브리핑’ 완료 토글)

- CTA 일관 삽입(블로그/쇼츠/리뷰)
  - 블로그/쇼츠에 CTA 자동 삽입(마지막 문단·버튼 문안)
  - SNS 캡션·해시태그 자동 생성(1클릭 복사)
  - 리뷰 답글 템플릿 5안(영수증/일반 리뷰용)

- 진행 관리(간단)
  - 업종별 ‘필수 5가지’ 한 페이지: 복사 / PDF / 관리자 바로가기
  - 체크 완료 저장 → 다음 접속 시 남은 항목만 보여주기
  - 콘텐츠 생성 흐름과 연동(CTA 자동 포함)

> 핵심: 고객에게 “뭘 적으라” 요구하지 않고, 필요한 산출물을 우리가 먼저 만들어 줍니다. 사장님은 복사/다운로드/바로가기만 누르면 됩니다.
