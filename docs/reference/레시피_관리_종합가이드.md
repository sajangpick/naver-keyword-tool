## �� 빠른 시작

### 1단계: Supabase 설정
```bash
# database/schemas/features/recipe/recipe-management-schema.sql 실행
```

### 2단계: 서버 실행
```bash
node server.js
```

### 3단계: 페이지 접속
```
http://localhost:3003/recipe-manager.html
```

### 4단계: AI 레시피 생성
- "AI 레시피 생성" 탭
- 재료 입력
- " AI 레시피 생성하기" 클릭

##  주요 기능

-  **검색**: 공공 API + 샘플 데이터
-  **AI 생성**: ChatGPT 활용
-  **원가 계산**: 자동 계산
-  **관리**: 저장/수정/삭제
- ⭐ **북마크**: 폴더별 저장
-  **평점**: 별점 및 리뷰
