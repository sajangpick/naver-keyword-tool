## 📋 프로젝트 개요

### 🎯 핵심 목표
**"식당 대표님들이 레시피를 검색하고, 만들고, 관리하는 올인원 시스템"**

### 💡 개발 원칙
1. **법적 안전성 최우선** - 공공데이터와 허가된 API만 사용
2. **핵심 기능 우선** - 검색, 생성, 관리 3대 핵심 기능에 집중
3. **싱글 페이지 앱(SPA)** - 페이지 이동 최소화로 사용성 극대화
4. **한국 식당 특화** - 밑반찬 관리, 메뉴 구성 등 한국 특유 기능
5. **실용성 중심** - 실제 식당 운영에 바로 적용 가능한 기능

## 🚀 개발 로드맵

### 📌 Phase 1: 핵심 기능 (즉시 시작) ✅

#### 1️⃣ 레시피 검색 시스템
```
✅ 구현 가능한 데이터 소스:
- 농촌진흥청 공공데이터 API (한식 9,000개)
- YouTube Data API v3 (레시피 영상)
- ChatGPT 레시피 생성 (실시간)

❌ 법적 위험으로 제외:
- 만개의레시피 크롤링
- 네이버 레시피 수집
- 기타 저작권 있는 사이트
```

#### 2️⃣ 나만의 레시피 생성
- **AI 레시피 생성기**: ChatGPT 활용
  - 재료 입력 → 레시피 자동 생성
  - 기존 레시피 변형 (매운맛↑, 단맛↓ 등)
  - 원가 자동 계산
- **레시피 에디터**
  - 재료/양념/조리과정 입력
  - 사진 업로드
  - 메모/팁 추가

#### 3️⃣ 레시피 관리 시스템
- **개인 레시피 보관함**
  - 폴더/카테고리 분류
  - 태그/검색 기능
  - 즐겨찾기
- **버전 관리**
  - 수정 이력 저장
  - 이전 버전 복원
- **백업/복구**
  - 자동 백업 (매일)
  - Excel/PDF 내보내기

### 📌 Phase 2: 한국 식당 특화 기능 (1개월 후)

  - 업종별 인기 메뉴 조합
  - 가격대별 구성 예시
- **AI 메뉴 추천**
  - 계절별 추천 메뉴
  - 타겟 고객별 구성

### 📌 Phase 3: 고급 기능 (3개월 후)

#### 6️⃣ 원가 관리 시스템
- 실시간 원가율 계산
- 재료 시세 반영 (공공데이터)
- 목표 마진율 설정

#### 7️⃣ 레시피 마켓플레이스
- **판매 옵션** (창업 전수)
  - 단품 레시피: 10만원~
  - 풀 패키지: 100만원~
  - 컨설팅 포함: 300만원~
- **수익 공유**: 플랫폼 30%, 판매자 70%

#### 8️⃣ 교육 플랫폼 연계
- 요리학원 파트너십
- 온라인 강의 연동
- 자격증 과정 안내

### Database Schema
```sql
-- 레시피 테이블
CREATE TABLE recipes (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES profiles(id),
  name VARCHAR(200) NOT NULL,
  category VARCHAR(50),
  ingredients JSONB,
  steps JSONB,
  cost_per_serving DECIMAL,
  prep_time INTEGER,
  difficulty VARCHAR(20),
  is_public BOOLEAN DEFAULT false,
  version INTEGER DEFAULT 1,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 밑반찬 매칭 테이블
CREATE TABLE side_dish_pairings (
  id UUID PRIMARY KEY,
  main_dish VARCHAR(100),
  side_dishes JSONB,
  user_id UUID REFERENCES profiles(id)
);
```
