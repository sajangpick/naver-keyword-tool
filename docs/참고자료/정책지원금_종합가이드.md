##  빠른 시작

### 1단계: 페이지 접속
```
http://localhost:3003/policy-support.html
```

### 2단계: 조건 입력
- 업종 선택
- 운영 기간
- 직원 수

### 3단계: 추천 정책 확인
- 자동으로 맞는 정책 추천
- 북마크 또는 신청

---

##  6가지 주요 정책

1. **소상공인 정책자금** - 최대 2억원
2. **스마트상점 지원금** - 최대 3천만원
3. **백년가게 지원금** - 인테리어 지원
4. **착한가격업소 지원금** - 월 2-5백만원
5. **노란우산 공제금** - 퇴직금 공제
6. **일자리 안정자금** - 신규 직원 고용
