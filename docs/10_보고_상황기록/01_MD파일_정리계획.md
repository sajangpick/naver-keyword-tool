## 정리 원칙

1. **기능별 그룹화**: 관련된 문서끼리 묶기
2. **네이밍 규칙**: 대문자로 통일 (예: GUIDE, SETUP)
3. **중복 제거**: 동일 내용은 하나로 통합
4. **버전 관리**: archive 폴더에 이전 버전 보관
5. **README 정리**: 각 폴더에 간단한 README
