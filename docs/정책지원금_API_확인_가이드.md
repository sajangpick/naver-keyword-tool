# 정책지원금 API 확인 가이드

## 현재 문제

API 테스트 결과 모든 엔드포인트가 실패했습니다:
- `data.go.kr` API: HTTP 404 "api not found"
- `odcloud.kr` API: HTTP 400 "등록되지 않은 서비스 입니다"

## 해결 방법

### 1. 공공데이터포털에서 실제 API 확인

1. **공공데이터포털 접속**: https://www.data.go.kr
2. **로그인** (회원가입 필요)
3. **검색창에서 검색**:
   - "중소벤처기업부 사업공고"
   - "중소기업 지원사업"
   - "소상공인 정책"
   - "정책지원금"
4. **API 선택 후 활용신청**:
   - 원하는 API 클릭
   - "활용신청" 버튼 클릭
   - 사용 목적 작성 (예: "소상공인 정책 정보 제공 서비스")
   - 승인 대기 (보통 즉시 ~ 1일)
5. **API 정보 확인**:
   - 승인 후 API 상세 페이지에서 확인:
     - **API ID** (예: 1421000)
     - **엔드포인트 URL**
     - **필수 파라미터**
     - **응답 형식** (XML/JSON)

### 2. API 키 확인

1. **마이페이지 > 개발계정** 이동
2. **발급받은 API 키 확인**:
   - 현재 사용 중인 키: `e45b26951c63da01a0d82653dd6101417c57f3812905e604bb4f60f80157bac8`
   - 이 키가 어떤 API에 등록되어 있는지 확인
3. **API별 등록 상태 확인**:
   - 각 API의 "활용신청 현황"에서 승인 여부 확인

### 3. 올바른 API 엔드포인트 찾기

공공데이터포털에서 확인한 정보:
- **API ID**: 예) 1421000, 1230000 등
- **서비스명**: 예) getBizPblancList, getBidPblancListInfo 등
- **엔드포인트**: 예) `https://apis.data.go.kr/{API_ID}/{서비스명}`

### 4. 코드 수정 필요 사항

공공데이터포털에서 확인한 실제 API 정보를 알려주시면:
1. `api/fetch-real-policy-data.js`의 API 엔드포인트 수정
2. `api/test-policy-api.js`의 테스트 엔드포인트 수정
3. 올바른 파라미터 설정
