### 2025.11.09 - 쇼츠 에디터 V1 공개 & 전자책 메뉴 노출

**배경**:
- 새로 제작한 `shorts-editor.html`과 `sajangpick-book.html`이 실제 서비스 메뉴에서 보이지 않아 사용자 접근성 저하
- 쇼츠 에디터에 대한 기능 확장 사항(권장 시간 커스터마이즈, 비용 경고, 프리미엄 기획/대본, Supabase 히스토리 저장) 문서 및 코드 반영 필요

**작업 내용**:
- `assets/common-header.js` 전역 네비게이션에 `shorts-editor.html`·`sajangpick-book.html` 링크 추가
- `index.html` 중앙 메뉴와 빠른 접근 메뉴에 영상편집·전자책 버튼 배치
- `shorts-editor.html` UI 업데이트: 권장 시간 직접 수정(5~60초 범위) + 길이 증가 시 비용 경고 문구 표기
- 전문가 의뢰 모드에 키워드 기반 “AI 기획 & 대본 초안” 생성 UI 구성, 히스토리 저장/재사용/삭제 인터랙션 정비
- `/api/shorts/plan-and-script` 엔드포인트 및 Supabase 연동 로직(저장/조회/삭제) 구현 및 테스트
- 관련 매뉴얼(`docs/01_기능_사용설명서/09_동영상편집기쇼츠.md`) 보강 계획 수립

**결과**:
- 쇼츠 에디터와 전자책 페이지가 모든 화면 상단 메뉴에서 접근 가능
- 셀프 제작 모드의 시간 조정 및 비용 경고로 사용자가 비용 구조를 직관적으로 인지
- 프리미엄 모드에서 AI 기획·대본 초안 생성 이력 관리 가능 (Supabase 장기 보관)
- 문서 체계 분리(동영상 전용 vs 전체 기록) 방향에 대해 사용자 합의 완료
