# ✍️ 블로그작성 기능 가이드

> 최종 업데이트: 2025년 11월 2일  
> 기능명: 블로그작성 (AI 블로그 자동 생성)

---

##  기능 개요

**블로그작성**은 ChatGPT, Gemini, Claude 3단계 AI를 활용하여 고품질 블로그 포스팅을 자동 생성하는 기능입니다.

###  핵심 기능
- 다단계 AI 블로그 생성 (3개 AI 활용)
- 가게 정보 자동 입력
- 네이버 플레이스 크롤링
- AI 키워드 추천
- 다양한 글쓰기 앵글 지원
- 블로그 초안 저장 및 수정

###  대상 사용자
- 식당/카페 대표
- 블로거
- 마케팅 담당자

---

##  기술 스택

### 프론트엔드
- `Blog-Editor.html` - 에디터 인터페이스
- 3개 탭 구성 (우리매장, 추천메뉴, 식재료)
- 드래그&드롭 파일 업로드
- 실시간 미리보기

### 백엔드
- Express.js 서버
- 3개 AI API (ChatGPT, Gemini, Claude)
- Puppeteer (네이버 플레이스 크롤링)
- Supabase (블로그 저장)

### AI 모델
- GPT-4o - 고품질 콘텐츠 생성
- Gemini Pro - 다양한 스타일 생성
- Claude 3 - 최종 검수 및 개선

##  API 엔드포인트

### 1. 블로그 생성 (3단계)
**POST** `/api/chatgpt-blog`

#### Step 1: 플레이스 크롤링
```json
{
  "step": "crawl",
  "placeUrl": "https://place.naver.com/..."
}
```

#### Step 2: 메뉴 분석
```json
{
  "step": "analyze",
  "placeInfo": { /* Step 1 결과 */ }
}
```

#### Step 3: 블로그 생성
```json
{
  "step": "generate-blog",
  "placeInfo": { /* 가게 정보 */ },
  "writingAngle": "첫방문_고객" or "단골_고객"
}
```

### 2. AI 키워드 추천
**POST** `/api/chatgpt-blog`

```json
{
  "step": "recommend-keywords",
  "storeName": "홍콩반점",
  "menuItems": ["짜장면", "탕수육"]
}
```

응답:
```json
{
  "success": true,
  "keywords": [
    "지역키워드: 강남 짜장면, 서울 중국식당",
    "음식키워드: 숟가락질, 찰진면발",
    "특징키워드: 저렴한 가격, 양많은, 친절한",
    "상황키워드: 회식메뉴, 무한리필불가",
    "SEO키워드: 맛집추천, 후기, 먹어본"
  ]
}
```

##  사용 방법

### 1. 페이지 접속
```
http://localhost:3003/Blog-Editor.html
```

### 2. 3개 탭 모드

####  탭 1: 우리매장
- 네이버 플레이스 URL 입력
- "플레이스 크롤링" 버튼 클릭
- 자동으로 가게 정보 입력됨
- "우리매장 소개 생성" 클릭

####  탭 2: 추천메뉴
- 대표 메뉴 입력
- "AI 메뉴 소개 생성" 클릭
- ChatGPT가 멋진 설명 작성

####  탭 3: 식재료
- "식재료 이야기" 주제 선택
- AI가 신선함/건강함 강조

### 3. AI 키워드 추천
- " AI 키워드 추천받기" 버튼 클릭
- 12개 다양한 키워드 생성
- 클릭으로 선택/해제
- 선택된 키워드 자동 입력

### 4. 글 생성 및 저장
- 3개 탭 모두 입력 완료
- "블로그 글 생성" 클릭
- 생성된 글 확인
- "발행하기" or "임시저장"

##  블로그 생성 프로세스

```
Step 1: 플레이스 크롤링
├─ 가게명, 주소, 영업시간 수집
├─ 전화번호, 메뉴 정보 추출
└─ 사진 다운로드

Step 2: 메뉴 분석
├─ 주 메뉴 파악
├─ 가격대 분석
└─ 특징 추출

Step 3: 3단계 AI 생성
├─ ChatGPT → 핵심 콘텐츠
├─ Gemini → 다양한 스타일
└─ Claude → 최종 검수 및 개선
```

##  글쓰기 앵글 (Writing Angle)

### 8가지 앵글
1. **첫방문 고객** - 신기하고 설레는 톤
2. **단골 고객** - 친근하고 정감있는 톤
3. **음식 평론가** - 전문적이고 자세한 톤
4. **바쁜 직장인** - 간결하고 실용적인 톤
5. **가족 단위** - 따뜻하고 포근한 톤
6. **데이트 커플** - 낭만적이고 감성적인 톤
7. **건강관심층** - 영양가 중심의 톤
8. **가성비 추구자** - 경제성 중심의 톤

##  요금 및 토큰

### 토큰 소비
- 블로그 생성: 약 2,000~3,000 토큰
- AI 키워드: 약 500 토큰
- 메뉴 분석: 약 800 토큰

### 구독별 한도
- 무료: 월 3회
- 베이직: 월 30회
- 프로: 무제한
