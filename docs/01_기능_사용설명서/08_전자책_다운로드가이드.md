### **Step 1️⃣: 사장픽 접속 및 로그인**
```
1. https://www.sajangpick.co.kr 접속
2. 우측 상단 "로그인" 버튼 클릭
3. 이메일 또는 카카오로 로그인
```

### **Step 2️⃣: 전자책 페이지 이동**
```

- 좌측 사이드바 하단 "📚 전자책" 아이콘 클릭

방법 B) 직접 URL 입력
- https://www.sajangpick.co.kr/sajangpick-book.html
```

### **Step 3️⃣: 무료 다운로드 버튼 클릭**
```
1. 전자책 페이지의 주황색 "📥 전자책 무료 다운로드" 버튼 클릭
2. PDF가 새 창에서 자동 열림
3. 이메일로도 다운로드 링크 발송 예정
```
