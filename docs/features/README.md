#  사장픽 전체 기능 가이드

> 최종 업데이트: 2025년 11월 2일

---

##  7가지 핵심 기능

### 1️⃣  키워드검색 (기능_키워드검색_가이드.md)
**네이버 키워드 검색 API를 활용한 마케팅 분석**
- 월간 검색수 조회
- 경쟁정도 분석
- 트렌드 파악
- **대상**: 마케터, 블로거, SEO 담당자

### 2️⃣  리뷰작성 (기능_리뷰작성_가이드.md)
**AI 기반 고객 리뷰 답글 자동 생성**
- 감정분석 (긍정/부정/중립)
- AI 답글 자동 생성
- 평판 관리
- **대상**: 식당/카페 운영자

### 3️⃣ ✍️ 블로그작성 (기능_블로그작성_가이드.md)
**3단계 AI를 활용한 고품질 블로그 생성**
- ChatGPT + Gemini + Claude 조합
- 네이버 플레이스 크롤링
- AI 키워드 추천
- **대상**: 블로거, 식당 대표

### 4️⃣  ChatGPT채팅 (기능_ChatGPT채팅_가이드.md)
**24시간 AI 비서와 대화**
- 식당 경영 상담
- 마케팅 전략 수립
- 비즈니스 문제 해결
- **대상**: 모든 사용자

### 5️⃣  정보게시판 (기능_정보게시판_가이드.md)
**식당 대표님들을 위한 정보 커뮤니티**
- 정책/법규
- 경영 노하우
- 트렌드/뉴스
- AI 뉴스 추천
- **대상**: 소상공인, 식당 운영자

### 6️⃣  정책지원금 (기능_정책지원금_가이드.md)
**소상공인 지원금 정보 한곳에서 관리**
- 6가지 주요 정책
- 자동 맞춤 추천
- 신청 절차 안내
- **대상**: 신규 창업자, 소상공인

### 7️⃣  레시피관리 (기능_레시피관리_가이드.md)
**AI 레시피 생성 및 원가 관리 시스템**
- ChatGPT 레시피 생성
- 자동 원가 계산
- 버전 관리
- 북마크 및 평점
- **대상**: 식당/카페 운영자

---

##  각 기능의 학습 순서

### 초급 (처음 시작)
1. 정보게시판 - 유용한 정보 학습
2. ChatGPT채팅 - AI 활용 경험
3. 키워드검색 - 마케팅 분석

### 중급 (마케팅 강화)
4. 블로그작성 - 콘텐츠 제작
5. 리뷰작성 - 평판 관리

### 고급 (운영 최적화)
6. 정책지원금 - 자금 확보
7. 레시피관리 - 메뉴 개발
