# 🔄 서버 재시작 가이드

## 현재 서버가 실행 중인지 확인

서버가 실행 중이면 **검은색 창(명령 프롬프트)** 또는 **VS Code 터미널**이 열려있고, 다음과 같은 메시지가 보입니다:
```
🚀 통합 API 서버가 시작되었습니다!
🌐 서버 주소: http://0.0.0.0:3003
```

---

## ✅ 방법 1: 터미널에서 재시작 (가장 간단)

### 1단계: 서버 종료

**서버가 실행 중인 검은색 창** 또는 **VS Code 터미널**에서:

1. **`Ctrl + C` 키를 누르세요**
   - 키보드에서 `Ctrl` 키와 `C` 키를 동시에 누름
   - 서버가 종료되면 커서가 다시 나타납니다

### 2단계: 서버 다시 시작

같은 터미널 창에서:
```bash
node server.js
```

**또는** (배치 파일 사용)
```bash
서버시작.bat
```

---

## ✅ 방법 2: 배치 파일 사용

### 1단계: 기존 서버 종료

1. 서버가 실행 중인 검은색 창을 찾아서 `Ctrl + C` 눌러 종료
2. 또는 창을 그냥 닫기

### 2단계: 배치 파일 실행

1. 프로젝트 폴더를 **Windows 탐색기**에서 엽니다
2. `서버시작.bat` 파일을 **더블클릭**
3. 새 검은색 창이 열리면서 서버가 시작됩니다

---

## ✅ 방법 3: VS Code에서 재시작

### 1단계: 터미널 열기

1. VS Code 하단에 **"터미널"** 또는 **"Terminal"** 탭 클릭
2. 또는 키보드에서 **`Ctrl + ` `** (백틱 키) 누르기

### 2단계: 서버 종료

터미널에서:
1. `Ctrl + C` 키를 누르세요

### 3단계: 서버 다시 시작

터미널에서:
```bash
node server.js
```

---

## 🔍 서버가 제대로 재시작되었는지 확인

서버가 성공적으로 시작되면 다음과 같은 메시지가 보입니다:

```
==========================================
🚀 통합 API 서버가 시작되었습니다!
🌐 서버 주소: http://0.0.0.0:3003
🏥 서버 상태: http://0.0.0.0:3003/health
📊 환경: development
==========================================
```

그런 다음 브라우저에서:
```
http://localhost:3003/ChatGPT.html
```
로 접속하여 정상 작동하는지 확인하세요.

---

## ⚠️ 주의사항

### 서버를 재시작할 때마다

1. **서버를 실행한 터미널/창을 닫지 마세요!**
   - 창을 닫으면 서버가 종료됩니다
   - 서버를 사용하는 동안은 **반드시 열어둬야** 합니다

2. **변경사항이 적용되지 않으면**
   - 서버를 완전히 종료한 후 다시 시작하세요
   - 브라우저를 새로고침(`F5`)해보세요

3. **포트가 이미 사용 중이라는 오류가 나면**
   - 이전 서버가 아직 실행 중일 수 있습니다
   - 모든 터미널 창을 확인하고 종료한 후 다시 시도하세요

---

## 💡 빠른 요약

1. **서버 종료**: `Ctrl + C`
2. **서버 시작**: `node server.js` 또는 `서버시작.bat` 더블클릭
3. **확인**: 터미널에 "서버가 시작되었습니다" 메시지 확인
4. **테스트**: 브라우저에서 `http://localhost:3003/ChatGPT.html` 접속

---

## 🆘 문제 해결

### "포트가 이미 사용 중입니다" 오류

**원인**: 이전 서버가 아직 실행 중

**해결**:
1. 모든 명령 프롬프트 창을 확인
2. 서버가 실행 중인 창에서 `Ctrl + C` 눌러 종료
3. 다시 시도

### "node를 찾을 수 없습니다" 오류

**원인**: Node.js가 설치되지 않음

**해결**: [Node.js 다운로드](https://nodejs.org/) 후 설치

### 서버는 시작되는데 연결이 안 됨

**해결**:
1. 서버가 정상 시작되었는지 확인 (터미널 메시지 확인)
2. 브라우저 주소 확인: `http://localhost:3003/ChatGPT.html`
3. 브라우저 새로고침(`F5`)
