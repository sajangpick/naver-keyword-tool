## 🆘 문제 해결

### "사이트맵을 읽을 수 없음" 오류

**원인**: sitemap.xml 파일에 접근 불가

**해결**:

1. 브라우저에서 직접 접속:
   ```
   https://sajangpick.vercel.app/sitemap.xml
   ```
2. 파일이 정상적으로 보이는지 확인
3. 안 보이면 배포 확인 필요

### "색인 생성 안 됨" 상태

**원인**: 구글/네이버 봇이 아직 방문하지 않음

**해결**:

1. 시간 기다리기 (1~7일)
2. URL 검사 도구로 수동 요청
3. robots.txt에서 크롤링 허용 확인

### 검색 결과에 안 나옴

**원인**: 색인은 됐지만 순위가 낮음

**해결**:

1. 브랜드 키워드로 검색 (예: "사장픽")
2. site: 명령어로 확인
3. SEO 최적화 작업 진행
