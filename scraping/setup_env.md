# 📝 .env 파일 설정 가이드

## 기존 .env 파일에 추가할 내용

프로젝트 루트의 `.env` 파일을 열고 아래 내용을 **추가**하세요:

```env
# ==================== ADLOG 스크래핑 설정 ====================
# 내 식당 정보 (네이버 플레이스에 등록된 정확한 이름)
MY_RESTAURANT_NAME=우리식당이름

# 추적할 식당 (여러 개 가능, 쉼표로 구분)
TRACK_RESTAURANTS=우리식당,경쟁사A,경쟁사B

# 스크래핑 설정
SCRAPING_DELAY=2  # 요청 간 딜레이 (초)
MAX_RETRIES=3     # 실패 시 재시도 횟수
```

## ✅ 이미 있는 환경변수들

기존 `.env`에 이미 있는 것들:
- `SUPABASE_URL` ✅
- `SUPABASE_SERVICE_KEY` ✅ 
- `OPENAI_API_KEY` ✅
- 기타 API 키들 ✅

## 🎯 설정 예시

```env
# 실제 예시
MY_RESTAURANT_NAME=BBQ치킨 강남점
TRACK_RESTAURANTS=BBQ치킨 강남점,교촌치킨 강남역점,굽네치킨 강남점
```

## 📌 주의사항

1. **정확한 식당 이름 사용**
   - 네이버 플레이스에 등록된 정확한 이름 입력
   - 띄어쓰기, 지점명까지 정확히

2. **한글 인코딩**
   - UTF-8로 저장되어야 함
   - 한글 깨짐 현상 주의

3. **보안**
   - `.env` 파일은 절대 Git에 올리지 마세요
   - `.gitignore`에 이미 포함되어 있음
