# 🚨 서버 실행 안내 (단계별)

## 현재 상황
- 브라우저에서 `localhost:3003`에 연결할 수 없음
- 서버가 실행되지 않은 상태

## ✅ 해결 방법 (3가지 중 선택)

### 방법 1: VS Code 터미널에서 직접 실행 (가장 확실)

1. **VS Code 하단 터미널 탭 클릭** (또는 `Ctrl + `` `)
2. 터미널에서 다음 명령어를 **순서대로** 입력:

```bash
# 1. Node.js 설치 확인
node --version

# 2. 패키지 설치 (처음 한 번만)
pnpm install

# 3. 서버 실행
node server.js
```

3. 서버가 시작되면 다음과 같은 메시지가 보입니다:
```
🚀 통합 API 서버가 시작되었습니다!
🌐 서버 주소: http://0.0.0.0:3003
```

4. **이 터미널 창을 열어둔 채로** 브라우저에서 접속:
```
http://localhost:3003/ChatGPT.html
```

### 방법 2: Windows 탐색기에서 배치 파일 실행

1. **Windows 탐색기** 열기 (VS Code 닫지 말고)
2. 프로젝트 폴더로 이동:
   `C:\Users\오다쉐프\Desktop\사징픽_권순미 최종수정중`
3. `서버시작.bat` 파일을 **더블클릭**
4. 검은색 창(명령 프롬프트)이 열리면서 서버가 시작됩니다
5. **이 창을 닫지 말고** 브라우저에서 접속:
```
http://localhost:3003/ChatGPT.html
```

### 방법 3: 새로운 명령 프롬프트 창에서 실행

1. **Windows 키 + R** 누르기
2. `cmd` 입력하고 Enter
3. 다음 명령어 입력:
```cmd
cd "C:\Users\오다쉐프\Desktop\사징픽_권순미 최종수정중"
node server.js
```

## 🔍 문제 해결

### "node를 찾을 수 없습니다" 오류

**원인:** Node.js가 설치되지 않음

**해결:**
1. [Node.js 공식 사이트](https://nodejs.org/)에서 다운로드
2. LTS 버전 설치 (18.x 이상)
3. 설치 후 컴퓨터 재시작
4. 다시 시도

### "모듈을 찾을 수 없습니다" 오류

**원인:** 패키지가 설치되지 않음

**해결:**
```bash
pnpm install
```
또는
```bash
npm install
```

### "포트가 이미 사용 중입니다" 오류

**원인:** 다른 프로그램이 3003 포트 사용 중

**해결:**
1. 다른 터미널 창을 모두 닫기
2. 또는 `.env` 파일에 다음 추가:
```
PORT=3004
```
3. 그런 다음 브라우저에서 `http://localhost:3004/ChatGPT.html` 접속

### 서버는 시작되는데 연결이 안 됨

**확인사항:**
1. 서버가 정상 시작되었는지 확인 (터미널 메시지 확인)
2. 브라우저 주소가 정확한지 확인:
   - ✅ `http://localhost:3003/ChatGPT.html`
   - ❌ `http://127.0.0.1:5501/...` (Live Server 사용 중)
3. 서버를 실행한 터미널 창이 열려있는지 확인
